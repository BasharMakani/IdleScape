# IdleScape

> A mobile idle clicker inspired by classic RuneScape.

🚧 **Status:** In development - final deadline May 7, 2026

📱 **Platform:** Mobile (target platform TBD)

---

## About

IdleScape is an idle clicker where each skill has its own dedicated mini clicker game. Players grind skills to gain XP and gold, then spend gold in the shop on gear, either for vanity or meaningful gameplay buffs. The endgame is maxing every skill.

The game features three core interfaces:

- **Skill Interface** - the main clicker view; click to gain XP and gold
- **Inventory** - equip gear and manage your loadout
- **Shop** - purchase tools, armor, and cosmetics

---

## Planned Features

**MVP (must-haves):**
- Woodcutting minigame
- Mining minigame
- Skill XP and leveling system
- Gold economy
- Shop with functional gear
- Inventory and equip interface
- Rarity system for objectives
- Basic sound design

**Stretch goals:**
- Combat skill with mob fights
- Fishing skill
- Boss drops + pet system
- Day-night cycle shader
- Full ambient audio and music
- Vanity cosmetics
